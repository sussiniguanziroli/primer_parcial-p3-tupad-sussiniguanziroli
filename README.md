# Food Store - Primer Parcial Programación III

## Descripción
Aplicación frontend e-commerce. Catálogo de productos y carrito de compras con persistencia en localStorage. Desarrollado con HTML, CSS y TypeScript sobre Vite. 
Sin frameworks.

## Alumno
Patricio María Sussini Guanziroli

## Funcionalidades
- Catálogo dinámico.
- Búsqueda por nombre.
- Filtrado por categoría.
- Carrito con persistencia local.
- Sumar, restar y eliminar productos del carrito.
- Cálculo de subtotal y total.

## Requisitos
- Node.js
- pnpm (recomendado)

## Instalación y Ejecución
1. Clonar repositorio:
   ```bash
   git clone https://github.com/sussiniguanziroli/primer_parcial-p3-tupad-sussiniguanziroli
   ```
2. Instalar dependencias:
   ```bash
   pnpm install
   ```
3. Iniciar servidor de desarrollo:
   ```bash
   pnpm dev
   ```
4. Abrir navegador en `http://localhost:5173`
